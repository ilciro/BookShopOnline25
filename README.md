# BookShopOnline25

# italian
# questa applicazione permette di acquistare un oggetto a scelta tra libro giornale e rivista, permettendone l'acquisto con pagamento in contrassegno o carta credito. Inoltre è possibile modificare sia il profilo dell'utente se registrato, sia gli acquisti effettuati da lui stesso. Se invece  è l'admin ad accedere oltre alla gestione degli utenti, è possibile generare anche il report delle vendite

# english
# This application allows you to purchase an item of your choice, whether it's a newspaper or a magazine, with payment by cash on delivery or credit card. You can also edit your registered user profile and your purchases. If the admin is logged in, you can manage users and generate sales reports.
